・導入方法
　Quest版モデルにはVRChat/Mobile/Toon Litが使用されています。
　VRCSDK2を導入で来ていない場合は先にVRCSDKを導入してください。

　Unityの「Assets」メニューから
　「Import package → Custom Package」でフォルダに入っている
「VRCAvatarModel_428chan＿Ver2018.unitypackage」を選んでください。

　設定済の「Prefab_428chan_PC.prefab」または「Prefab_428chan_Quest」の使用を推奨します。
　「Prefab_Mini428chan」はおまけの設定済みPrefabとなっています。
　
・FBXについて
　「VRC_428chan_PC.fbx」「VRC_428chan_quest.fbx」はメッシュが一つに統合されたデータとなります。
　「VRC_428chan_PC_ALLMesh.fbx」は各パーツにメッシュが切り分けられた状態のデータとなります。
　「VRC_428chan_quest_ALLEarMesh.fbx」は耳パーツを選べるように切り分けられたデータとなります。
　「256pory428chan.fbx」は256ポリゴンで作られた428ちゃんのオマケデータとなります。
　
　マテリアル設定はお好みで、別のシェーダーの利用も可能です。
　ペイントツールSAI、CLIPStudioなどのレイヤー情報を編集できるペイントツールで
　psdファイルのテクスチャ改変ができます。
　基本的に「☆」がついたレイヤーは塗りつぶしのみで色変えが可能です。
　（「☆靴色」、「☆布色」のみ白色の部分と赤色の部分が分かれていません）

　Ver2までのテクスチャの互換性はありません、ご了承ください。
